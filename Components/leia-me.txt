O código espiral.c foi criado no ambiente computacional code blocks e deve ser executado com um compilador de linguagem c

Instruções:

O arquivo de eventos não pode conter caracteres como 'ç', letras acentuadas ou símbolos especiais.
Apenas números e letras (maiúsculas e minúsculas) do alfabeto normal (26 letras) e caracteres como '-', '_', '!' e '?'
Deve-se existir uma linha em branco (e apenas uma) após o último evento do arquivo de eventos .txt

Os dados devem ser inseridos corretamente para o bom funcionamento do programa
O endereço do arquivo de eventos deve ser inserido da seguinte forma: "C:\Users\danie\Desktop\Evento.txt"
ou simplesmente "Evento.txt" caso este se encontre no diretório do programa

Recomenda-se colocar o navegador como aplicativo padrão para abrir arquivos .pdf

Certifique-se que TODOS os componentes do arquivo .zip estejam na mesma pasta

Adicione os arquivos pdfgen.h, pdfgen.c, Funcoes.c e Biblioteca.h no projeto do seu IDE
